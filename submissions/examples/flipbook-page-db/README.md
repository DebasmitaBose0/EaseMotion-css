# 3D Flip Book Page Component

1. What does this do? Implements a realistic 3D page turning transition inside a mock book container using pure CSS transforms.
2. How is it used? Place the `.book` layout with static pages and `.page-wrapper` containing the flipping page label in your layout.
3. Why is it useful? It illustrates how to leverage `transform-origin` (set to `left center`) alongside `rotateY` to mimic realistic 3D paper movement without script calculations.
